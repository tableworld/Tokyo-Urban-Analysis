# Tokyo Project | SYNTHARA ARC Urban Systems Prototype

## Vision
Reimagining Tokyo’s urban data through a design philosophy that blends data science, aesthetic urbanism, and generative foresight.

## Current Features
- Population density vs. commute time plotted for key wards.
- Green space represented as scaled visual elements.
- Visualizes trade-offs in urban design using real-world data structures.

## Next Steps
- Integrate real, updated Tokyo data.
- Expand to interactive visualizations (Plotly, Altair).
- Explore predictive modeling for green space optimization.
- Develop a meta-design narrative aligned with SYNTHARA principles.
